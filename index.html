Ajuda

Esse sistema irá te ajudar a aprender as palavras que você não sabe ou está aprendendo, para que isso seja possível, primeiro ele precisa entender quais palavras você já sabe. Nessa fase de 'amadurecimento', você verá palavras que já conhece, caso conheça realmente determinada palavra, você poderá marcar como 'Já sei', e ela não aparecerá mais na tabela de traduções.


Você poderá também eliminar várias palavras de uma vez, usando o 'Eliminar palavras'. Você deve colocar o número de palavras que quer eliminar, e o sistema irá eliminar as primeiras X palavras mais faladas no Inglês. Ex: Colocando o número 250, o sistema irá eliminar as 250 palavras mais usadas no Inglês

Eliminar Palavras

Você deve colocar o número de palavras que quer eliminar, e o sistema irá eliminar as primeiras X palavras mais faladas no Inglês - segundo o site blablabla.com Ex: Colocando o número 250, o sistema irá eliminar as 250 palavras mais usadas no Inglês.
